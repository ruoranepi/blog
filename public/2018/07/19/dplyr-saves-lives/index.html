dplyr saves lives

  • July 19, 2018
  • Learn R  
  • R  
  • data science  

Transitioning from Stata to R is painful. So for over half a year, I was using base R for data cleaning and manipulation (double PAINFUL!), before finally having time to learn the many uses of the dplyr package. Life is so much simpler. All thanks go to YouTube videos from Prof. Roger Peng: timesaver and lifesaver!
