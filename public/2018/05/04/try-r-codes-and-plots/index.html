Starting with R codes in tidyverse

Set up

I’m following the R for data science book to learn some better practices for organizing my analyses. My motivation for this is that I’ve only just realized that I have been using base R for most of my previous data cleaning works – maybe I should start learning to use packages that can simplify this process.

tidyverse may be the place to start (heard about it from Roger Peng and Hilary Parker’s podcast).

One important distinction is that each R project should be self-contained. Lessons learned form this post on project-oriented workflow:

  • Use relative paths, and do not setwd().
  • Open new sessions for each research question, and do not rm(list=ls()).
